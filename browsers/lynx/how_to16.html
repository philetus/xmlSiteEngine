[ xmlSE . how to . edit a browser ] edit

To edit a browser, go to the configure site page and follow the 'edit' link next to the browser version you want to edit.

If you want to change the name of the browser, you also have to go into the xmlSiteEngine/system/xml/ directory and change the name of the template file, so doing that is not recommended.

The rest of the fields are the same as when you create a new browser.
