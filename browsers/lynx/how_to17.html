[ xmlSE . how to . edit a category ] edit

To make changes to a category after you have created it, go to the configure site page. Follow the 'edit' link next to the category you want to edit.

Enter your name and password at the prompt. If you try to change the name of the category, the member title, or the member tyoe, you will also have to go and edit the category xml file in 'xmlSiteEngine/system/xml/' to reflect those changes, which is not recommended. You can change the 'user' subfield of the 'member' category, and as new records are created they will be assigned passwords.

The 'sort' field can still be modified like when you create a new category.
