[ xmlSE . how to . create a reference template ] edit

Reference templates control the appearance of references to a record in other records thet it mentions, as well as from the search results page. To create a new template, go to the configure site page, find the browser you want to create a reference template for, and follow the 'new template' link.

Enter your name and password at the prompt.

The name of the template should be 'reference' and the subcategory name should be the name of the category this is to be the reference template for.

The template can include any html suitable for this browser version of the template, as well as the template tags described in 'edit an html template'.
