[ xmlSE . how to . create a new subcategory ] edit

To create a new subcategory, go to edit a record, and type the name of the new subcategory in the 'name' subfield of the 'subcategory' field. Subcategories are automatically generated from the record subcategory fields, and all the records with the same subcategory name will be in the same subcategory.
